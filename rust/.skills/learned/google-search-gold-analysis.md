# Google Search - Gold/XAUUSD Analysis

## URL Pattern
google.com/search?q=XAU/USD+analysis OR gold+price+forecast

## Workflow
1. Navigate: `https://www.google.com/search?q=XAU/USD+today+analysis&hl=en&num=10`
2. Read Google AI Overview snippet (quick summary)
3. Click top 2-3 results: LiteFinance (daily forecast), City Index (weekly outlook), TradingView
4. Extract: price, trend, support/resistance, indicators, expert views

## Key Sources Found
- LiteFinance: https://www.litefinance.org/blog/analysts-opinions/gold-price-prediction-forecast/daily-and-weekly/
  - Daily/weekly forecast with precise S/R levels
  - Updated daily (H4 technical analysis)
- City Index: https://www.cityindex.com/en-uk/news-and-analysis/
  - Weekly gold outlook from Fawad Razaqzada
  - Macro + technical blend
- Trading Economics: https://tradingeconomics.com/commodity/gold
  - Real-time price data
- J.P. Morgan: https://www.jpmorgan.com/insights/global-research/commodities/gold-prices
  - Long-term institutional forecast
